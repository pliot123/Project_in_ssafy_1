## 기획 배경 

평균 매매가와 매매해약가 비교시 매매해약가가 비정상적으로 높을 경우, 가격 거품을 의심해볼 수 있도록 만들고 싶었습니다.

뿐만 아니라, 가격 비교를 통해 합리적인 부동산을 구매할 수 있길 바라는 마음에 만들게 되었습니다.

## 역할

* vue.js & spring & Mybatis 웹 프로젝트
* 자바 크롤링
* 카카오 map API
* 부동산 공공데이터 활용
* 이메일 인증 받기
